## ☕ Usando <Super Ensino>

Após a realização do clone do projeto. Para usar o projeto <Super Ensino>, siga estas etapas:

1º Instale as dependencias do Projeto

```sh
npm i
```
ou
```sh
yarn install
```

2º Inicie a Aplicação

```sh
npm rum dev
```
ou
```sh
yarn rum dev
```
